Import Ascents — {{ label }}

Format

Paste CSV data with two columns:

hill_name,ascent_dates
  • hill_name — exact name match (case-insensitive)
  • ascent_dates — one or more dates separated by ;
  • Dates accepted as YYYY-MM-DD or DD/MM/YYYY
  • Existing ascents for any hill in the import are replaced
  • A header row (hill_name,ascent_dates) is optional and will be skipped

Example

hill_name,ascent_dates
Ben Nevis,2023-07-15;2024-08-20
Cairn Gorm,2022-09-03
Buachaille Etive Mòr,01/06/2021;15/08/2023
